
# image-director

A CLI Application in Node.JS that can manipulate images




## Requirements
Python and Node.js
## Installation

```bash
git clone https://github.com/GerGerApex1/image-director.git
cd image-director
```

## Supported Operating Systems 
- [X]  Windows
- [ ]  MacOS
- [ ]  Linux
## Core Functions
### Modes
- [x] Image File Reducer
- [X] Image Rotater
### File Support
- [X] Singular File
- [X] Multiple Files
- [X] Directory
